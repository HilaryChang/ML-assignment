# Assignment 1

## Quick Start

```
$ pip install -r requirements.txt
$ python main.py train
```

## 調整參數

```
$ python main.py train --k=3 --distance-type=sad
```

## 超參數優化(Hyper-parameter optimization)

```
$ python main.py hyperparameter
```
